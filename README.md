### Rodando em ambiente local

**1:** Clone o repositório.

```
git clone https://github.com/bllackdev/ad-2019/
```

**2:** Acesse o diretório do projeto.

```
cd ad-2019/front-end
```

**3:** Execute. (Porta 3000)
```
yarn start
```
